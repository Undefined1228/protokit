| 구분 | Phase |
|------|-------|

---

## 목표

WebSocket 연결을 클라이언트·서버 양쪽에서 테스트.

---

## 체크리스트

### 클라이언트

- [ ] URL 입력 (ws:// / wss://)
- [ ] 연결 헤더 설정
- [ ] Connect / Disconnect 버튼
- [ ] 연결 상태 표시 (연결 중 / 연결됨 / 끊김)
- [ ] 메시지 전송: text, JSON
- [ ] 메시지 스트림 뷰어: timestamp · 방향(송신/수신) · payload 표시
- [ ] 메시지 필터링 (송신만 / 수신만 / 전체)
- [ ] 스트림 로그 저장

### 서버

- [ ] 포트 설정 후 로컬 WS 서버 시작 / 중지
- [ ] 연결된 클라이언트 목록 표시
- [ ] 수신 메시지 로그
- [ ] 메시지 전송: 특정 클라이언트 / 전체 broadcast
- [ ] 이벤트 핸들러 정의 (연결 · 메시지 · 종료 시 자동 응답 규칙)

---
