# Phase 2 — 컬렉션 & 환경변수

## 목표

요청을 저장·재사용하고, 환경별로 변수를 분리해 관리.

---

## 기능 범위

### 컬렉션

- 요청을 컬렉션(폴더) 단위로 저장
- 컬렉션 생성 / 이름 변경 / 삭제
- 요청 저장 / 이름 변경 / 삭제 / 순서 변경
- 사이드바 TreeView로 컬렉션·요청 목록 표시

### 환경변수

- 환경(Environment) 생성·전환 (예: dev, staging, prod)
- `{{변수명}}` 형태로 URL·Headers·Body에서 치환
- 변수 편집 UI
