# Phase 7 — SSE (Server-Sent Events)

## 목표

SSE 스트림을 클라이언트·서버 양쪽에서 테스트.

---

## 체크리스트

### 클라이언트

- [ ] URL 입력
- [ ] 연결 헤더 설정
- [ ] Connect / Disconnect 버튼
- [ ] 연결 상태 표시
- [ ] 수신 이벤트 스트림 뷰어: timestamp · event type · data · id 표시
- [ ] 이벤트 타입별 필터링
- [ ] 스트림 로그 저장
- [ ] 자동 재연결 동작 확인 (Last-Event-ID 추적)

### 서버

- [ ] 포트 설정 후 로컬 SSE 서버 시작 / 중지
- [ ] 연결된 클라이언트 목록 표시
- [ ] 이벤트 수동 전송: event type · data · id 설정
- [ ] 반복 전송 스케줄 설정 (예: N초마다 이벤트 자동 발행)
- [ ] 전체 클라이언트 broadcast

---

## 제외 범위 (추후 검토)

- 인증 연동 (SSE는 헤더 커스텀이 브라우저에서 제한적 — 서버 구현 시에만 해당)
